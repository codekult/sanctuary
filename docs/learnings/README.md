# Learnings

Documented solutions to problems encountered during development. This is the compounding part — every solved problem becomes searchable institutional knowledge.

## Format

Each learning should include:

- **Problem**: What went wrong or was hard?
- **What we tried**: Approaches that didn't work (and why)
- **Solution**: What worked
- **Key insight**: The takeaway for next time
